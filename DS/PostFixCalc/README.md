# Post Fix Calculator

## Overview
    C program that is a post fix calculator where the mathematical expression is represented in post fix notation. 
    
    For example: 1+2 is written as 12+

## Compilation
    make

## Usage
    run ./postfixC

## Known Limitations
    advanced post-fix expressions not supported
